# Running on Google Colab

1. **Open the Notebook**:
   - Go to [Google Colab](https://colab.research.google.com/).
   - Select "GitHub" tab.
   - Enter this repo URL and choose `notebooks/00_onboarding.ipynb`.

2. **Enable GPU**:
   - Go to **Runtime** > **Change runtime type**.
   - Select **T4 GPU**.
   - Click **Save**.

3. **Run All**:
   - Click **Runtime** > **Run all**.
   - Accept the warning about not authored by Google.

4. **Troubleshooting**:
   - If you see "No GPU detected", make sure step 2 was done and you see a GPU connected in the resources tab.
   - If imports fail, try **Runtime** > **Restart session** and run the cells again.
